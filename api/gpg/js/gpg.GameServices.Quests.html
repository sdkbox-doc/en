(static) ClaimMilestone(milestone_id, callback?)

Asynchronously claims the milestone. Doing so calls the server, marking the milestone as completed. If the milestone is currently unclaimable, or if it has been claimed already on this or another device, you will get an error. If this call returns QuestClaimMilestoneStatus.VALID, you (as a developer) must still reward the player. Use the milestone CompletionRewardData to do so.
