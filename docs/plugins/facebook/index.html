Facebook

SDKBOX Facebook plugin provides a comprehensive collection of Facebook's social features, giving your players the ability to share content with their friends and allowing you to create a personal, social gaming experience. With this addition, you can build cross-platform games by sharing a single code base and integrate Facebook features without importing additional frameworks or SDKs.
